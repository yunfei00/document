# 1 Excel中时间戳转化成指定格式的时间
1. Excel默认不支持Unix格式时间戳，这在导入数据时十分不便。可以用以下公式将时间戳转换成Excel格式的时间：
	```
	=(x+8*3600)/86400+70*365+19
	```
2. 公式说明
	```
	x      时间戳的单元格
	8*3600 8为中国的时区 
	```
3. 原理介绍
	
	Excel的日期实际上是序列值，它以1900-1-1为1，每过一天序列值加1。
	Unix时间戳是从1970-1-1 0:00:00 UTC开始到现在经过的秒数。
