# 1 特殊符号
符号	说明	编码（实际使用中需要去掉加号）
←	左箭头	&+larr;
→	右箭头	&+rarr;
↑	上箭头	&+uarr;
↓	下箭头	&+darr;
↔	水平箭头	&+harr;
↕	竖直箭头	&+varr;
⇐	双线左箭头	&+lArr;
⇒	双线右箭头	&+rArr;
⇑	双线上箭头	&+uArr;
⇓	双线上箭头	&+dArr;
⇔	双线水平双箭头	&+hArr;
⇕	双线竖直箭头	&+vArr;
