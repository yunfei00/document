# argparse 概览

## argparse 是什么
`argparse` 用于把命令行参数解析成 Python 对象（Namespace），并自动提供帮助信息与校验。

## 什么时候用
- 任何需要命令行入口的工具：训练/评估/数据生成/导出/部署脚本
- 内网/离线环境：不想引入 click/typer 等第三方依赖
- 需要可维护、可扩展的命令行接口

## 与其它方式对比
- `sys.argv`：简单但易乱，校验/帮助要自己写
- `argparse`：稳定、标准库、适合长期维护
- `click/typer`：体验好但有第三方依赖（对离线交付不友好）

## 一句话总结（工程视角）
> argparse 是“长期可维护 CLI 的最稳妥默认选择”。
